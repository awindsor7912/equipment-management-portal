# System Roadmap

## ✅ Phase 1: Foundation (Completed)
* **Centralized Registry:** Migration of regional spreadsheets to normalized Dataverse tables.
* **Model-Driven Interface:** Deployment of the primary administrative UI and core entity forms.
* **Relational Structure:** Established basic links between assets and maintenance logs.

## 🚀 Phase 2: Operational Maturity (In Progress)
* **Governance Hardening:** Implementation of duplicate detection rules and mandatory field enforcement.
* **Maintenance Module:** Advanced scheduled maintenance logic with automated reminder notifications via Power Automate.
* **Assignment Tracking:** Development of a "Custody" entity for check-in/check-out logic and historical audit trails.

## 🔭 Phase 3: Integration & Intelligence (Future)
* **Live SQL Integration:** Transitioning from static dataflows to real-time mirrored tables. Currently in sample prototyping phase. 
* **Predictive Analytics:** Power BI integration for maintenance trend analysis and asset utilization reporting.
* **Asset Tagging:** Implementation of QR/Barcode scanning for mobile asset identification.
